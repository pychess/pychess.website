News

Staunton 0.10 Final!

This release of PyChess has been on the way for more than two years. We have had a lot of last minute fixes since the release candidate. A few of them for bugs that have been around a long time. In particular there has been a lot of stabilization of CECP and UCI, so they should now work with an even wider set of engines. You can even run windows engines through wine.

Read the who announcement on the mailing list.

PyChess has a new website!

After a lot of great work from the community, in particular Andrew Nagi and Matthew Gatto, we can now pressent the new PyChess.org! We would also like to thank Alexis Flint for the domain.

More news should follow shortly, regarding the upcomming PyChess Stauton release.

Staunton 0.10 RC 1

2010/10/15: This is the only planned release candidate before 0.10 final. Lots of bugs fixed and the new seek dialog is more polished now. Please update your translations on Launchpad!
